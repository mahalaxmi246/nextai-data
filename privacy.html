Privacy Policy for NextAI Tab

Effective Date: September 2025

NextAI Tab is a lightweight Chrome extension that displays an AI tool of the day, a short joke, and provides a simple search box on your new tab page.

Information We Collect

We do not collect, store, or share any personally identifiable information (PII) or sensitive user data.

Permissions Used

  • storage – Used only for caching AI tools and jokes locally to improve loading performance. No personal data is stored.
  • host_permissions – The extension fetches static JSON files (AI tools and jokes) hosted on our GitHub Pages repository (https://mahalaxmi246.github.io/nextai-data/).

Search Behavior

NextAI Tab provides a search box that routes queries to the browser’s default search engine using the Chrome Search API. We do not intercept, record, or transmit search queries to any of our servers. No personal data or browsing history is collected by this extension.

How the Extension Works

  • Shows a randomly selected AI tool of the day with its logo, description, and official link.
  • Displays a light-hearted joke fetched from static JSON files.
  • Provides a search bar that respects your browser’s default search engine.
  • Chrome storage is used to cache JSON responses for up to 24 hours, then refreshed automatically.

Third-Party Services

  • No third-party analytics, tracking, or advertising services are used in this extension.
  • Logos of AI tools are loaded from publicly available sources (e.g., SimpleIcons CDN). These requests are direct image loads, and no tracking occurs.
  • An “Ad slot” placeholder is shown in the footer for design purposes. No real ads are loaded.

Data Sharing

No data is shared with us, third parties, or advertisers. All operations happen locally in your browser.

Security

Since we do not handle user data, there is no risk of leaks, profiling, or unauthorized access from this extension.

Policy Updates

We may update this privacy policy in the future if new features are added. Any changes will be reflected on this page.
